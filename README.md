# Inteligência Artificial Generativa Aplicada à Programação com Python

Este repositório foi criado para armazenar os projetos, códigos e anotações desenvolvidos durante o curso do **SENAI**. O objetivo principal é aprender a integrar o poder das IAs Generativas no ecossistema Python para otimizar o desenvolvimento de software.

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Python
* **IAs/Ferramentas:** (Ex: Chat-gpt,Gemini)

## 📚 Conteúdo do Curso
* [ ] Introdução à IA Generativa e Engenharia de Prompt
* [ ] Automação de código com Python
* [ ] Integração de APIs de LLMs
* [ ] Projetos práticos
